# User Guide

Welcome to Pocket Money! This guide covers everything you need to know to use the app.

## Getting Started

### Creating an Account

1. Visit the app and click **Register**
2. Enter your name, email, and password
3. Optionally enter an invite code if joining an existing family
4. Click **Create account**

**Without invite code**: A new family is created for you automatically.

**With invite code**: You join the existing family and can immediately see their children.

### First Steps

After registering:

1. **Add a child** - Click "Add Child" on the dashboard
2. **Make a deposit** - Open the child's profile and click "Add"
3. **Set up allowance** - Go to child settings to create recurring payments
4. **Invite family** - Go to Settings to generate an invite code

## Navigation

- **Dashboard** (`/`) - Overview of all children with balances
- **Child Profile** - Click a child to see transactions and balance history
- **Child Settings** - Gear icon on child profile for targets and recurring payments
- **App Settings** - Top-right menu for currency, PIN, and family management
- **Profile** - Update your name and photo

## Tips

### For Parents

- Set up recurring payments to automate allowance
- Use the transaction history to see who made each deposit/withdrawal
- Create saving targets to teach financial goals

### For Multiple Parents/Guardians

- Use invite codes to share access
- All family members can add/withdraw money
- Transactions show which parent made them
- Settings (currency, PIN) are shared per-family

### Mobile Use

The app is a Progressive Web App (PWA):

- Add to home screen for app-like experience
- Works offline for viewing (requires connection for changes)
- Touch-friendly interface
